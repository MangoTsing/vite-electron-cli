## What

This is a tool to quickly help users build simple Vite Vue3 TypeScript Electron applications.

If you want to try it, you can see the following:

## Install

`npm i vite-electron-cli -g`

## How

```

vitel init <project_name>

cd <project_name>

yarn or npm i

vitel dev

```
